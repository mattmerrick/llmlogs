üí∞ Cost Calculator

Calculate and compare costs across different models:

function calculateCost(model, inputTokens, outputTokens) {
    const pricing = model.versions[0].pricing;
    return (inputTokens * pricing.input + 
            outputTokens * pricing.output) / 1000;
}

üîç Model Finder

Find the best model for specific capabilities:

function findModelsWithCapability(data, capability) {
    return Object.entries(data.models)
        .filter(([_, model]) => 
            model.capabilities.includes(capability))
        .map(([name, model]) => name);
}
