X(Twitter) MCP server

An MCP server to create, manage and publish X/Twitter posts directly through Claude chat.

Quick Setup

Installing via Smithery

To install X(Twitter) MCP Server for Claude Desktop automatically via Smithery : 

npx -y @smithery/cli install x-mcp --client claude

Manual Installation

  1. Clone the repository: 
git clone https://github.com/yourusername/x-mcp.git
  1. Install UV globally using Homebrew in Terminal: 
brew install uv
  1. Create claude_desktop_config.json:
  2. For MacOS: Open directory ~/Library/Application Support/Claude/ and create the file inside it

  3. For Windows: Open directory %APPDATA%/Claude/ and create the file inside it

  4. Add this configuration to claude_desktop_config.json: 

{
  "mcpServers": {
    "x_mcp": {
      "command": "uv",
      "args": [
        "--directory",
        "/path/to/x-mcp",
        "run",
        "x-mcp"
      ],
      "env": {
        "TWITTER_API_KEY": "your_api_key",
        "TWITTER_API_SECRET": "your_api_secret",
        "TWITTER_ACCESS_TOKEN": "your_access_token",
        "TWITTER_ACCESS_TOKEN_SECRET": "your_access_token_secret"
      }
    }
  }
}
  1. Get your X/Twitter API credentials:
  2. Go to X API Developer Portal
  3. Create a project
  4. In User Authentication Settings: Set up with Read and Write permissions, Web App type
  5. Set Callback URL to http://localhost/ and Website URL to http://example.com/

  6. Generate and copy all keys and tokens from Keys and Tokens section

  7. Update the config file:

  8. Replace /path/to/x-mcp with your actual repository path

  9. Add your X/Twitter API credentials

  10. Quit Claude completely and reopen it

Troubleshooting

If not working: - Make sure UV is installed globally (if not, uninstall with pip uninstall uv and reinstall with brew install uv ) - Or find UV path with which uv and replace "command": "uv" with the full path - Verify all X/Twitter credentials are correct - Check if the x-mcp path in config matches your actual repository location
