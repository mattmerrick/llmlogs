WordPress Rules for Cursor Editor

1 Rule
CMS Platform
.mdc Format

About These Rules

These WordPress coding rules are specifically formatted for the Cursor Editor as .mdc files, focusing on WordPress 6+ features and modern theme development patterns. Each rule is designed to help you write more maintainable WordPress themes while leveraging Cursor's AI capabilities for enhanced development.

1. Block Theme Development and Patterns

Implement modern block theme development practices with proper template structure and block patterns. This approach demonstrates WordPress best practices including block templates, theme.json configuration, and block pattern registration.

wordpress-block-theme.mdc 
// theme.json
{
    "$schema": "https://schemas.wp.org/trunk/theme.json",
    "version": 2,
    "settings": {
        "color": {
            "palette": [
                {
                    "slug": "primary",
                    "color": "#0073aa",
                    "name": "Primary"
                },
                {
                    "slug": "secondary",
                    "color": "#23282d",
                    "name": "Secondary"
                }
            ],
            "gradients": [
                {
                    "slug": "primary-to-secondary",
                    "gradient": "linear-gradient(135deg, #0073aa 0%, #23282d 100%)",
                    "name": "Primary to Secondary"
                }
            ]
        },
        "typography": {
            "fontSizes": [
                {
                    "slug": "small",
                    "size": "0.875rem",
                    "name": "Small"
                },
                {
                    "slug": "medium",
                    "size": "1rem",
                    "name": "Medium"
                }
            ],
            "fontFamilies": [
                {
                    "fontFamily": "-apple-system,BlinkMacSystemFont,\"Segoe UI\",Roboto,Oxygen-Sans,Ubuntu,Cantarell,\"Helvetica Neue\",sans-serif",
                    "slug": "system",
                    "name": "System Font"
                }
            ]
        },
        "layout": {
            "contentSize": "800px",
            "wideSize": "1100px"
        }
    },
    "styles": {
        "blocks": {
            "core/button": {
                "color": {
                    "background": "var(--wp--preset--color--primary)",
                    "text": "#ffffff"
                },
                "typography": {
                    "fontSize": "var(--wp--preset--font-size--medium)"
                }
            }
        }
    }
}

// functions.php
 __('Primary Menu', 'modern-blocks'),
            'footer' => __('Footer Menu', 'modern-blocks'),
        ]);
    }

    public function register_block_patterns(): void {
        register_block_pattern(
            'modern-blocks/hero-section',
            [
                'title' => __('Hero Section', 'modern-blocks'),
                'description' => __('A hero section with image, heading, and CTA', 'modern-blocks'),
                'content' => $this->get_hero_pattern(),
                'categories' => ['header'],
            ]
        );
    }

    private function get_hero_pattern(): string {
        return '
        

            
            

                
                

                    
                    
Welcome to Modern Blocks

                    

                    
                    
Experience the future of WordPress theme development with our block-first approach.

                    

                    
                    

                        
                        
Get Started

                        
                    

                    
                

                

                
                

                    
                    
                    
                

                
            

            
        

        ';
    }

    public function enqueue_assets(): void {
        wp_enqueue_style(
            'modern-blocks-style',
            get_stylesheet_uri(),
            [],
            wp_get_theme()->get('Version')
        );

        // Add custom fonts
        wp_enqueue_style(
            'modern-blocks-fonts',
            'https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap',
            [],
            null
        );
    }
}

new ModernBlockTheme();

Why This Rule Matters

  • Ensures modern block theme development practices
  • Makes theme customization more maintainable
  • Improves reusability through block patterns
  • Enables better theme.json configuration
  • Makes theme development more scalable

Using with Cursor

Cursor's AI capabilities can help you:

  • Generate block patterns and templates
  • Suggest theme.json configurations
  • Identify opportunities for pattern extraction
  • Debug theme compatibility issues
  • Convert classic themes to block themes
