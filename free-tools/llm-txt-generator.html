What is LLMs.txt?

LLMs.txt is a configuration file that helps you control how Large Language Models (LLMs) like ChatGPT and Claude interact with your website's content. It allows you to:

  • Define content discovery rules for AI models
  • Set content priorities and training windows
  • Configure citation policies and preferences
  • Manage content freshness and updates
  • Protect sensitive content from AI training

This free tool helps you generate an optimized LLMs.txt file tailored to your website's needs.

LLMs.txt Best Practices

  • Set appropriate training windows for time-sensitive content
  • Configure clear content relationships for better context
  • Define citation policies to improve attribution
  • Use priority settings to highlight important content
  • Regularly update freshness settings for dynamic content
