Privacy Policy

Effective Date: [Insert date]

Thank you for using LLM SEO Checker — a Chrome extension that helps users assess whether their current webpage is optimized for visibility in large language models like ChatGPT, Claude, and Perplexity.

Data Collection

We do not collect, store, or transmit any personal data or browsing activity.

The LLM SEO Checker extension performs all functionality locally within your browser. It does not track, save, or send any information to external servers.

What We Access

The extension only accesses the content of the active tab when you manually activate it. This is used solely to:

  • Analyze the current page's structure (e.g., headings, metadata, semantic HTML)
  • Check for the presence of a llms.txt file
  • Provide an LLM SEO readiness score and suggestions

No Personally Identifiable Information

LLM SEO Checker does not collect or use any of the following:

  • Email addresses or names
  • IP addresses or GPS location
  • Passwords or authentication data
  • Financial, health, or other sensitive information

Third-Party Services

This extension does not use any third-party analytics, trackers, or external APIs. All functionality is self-contained.

Changes

We may update this policy to reflect improvements or changes to the extension. If we do, the updated policy will be posted here with a revised date.

Contact
